# Engineering ST – Session Summary

In this session, I learned the fundamentals of **Onshape Web** for 2D sketching and 3D modeling.

### Key Learning Outcomes

* Mastered the basic sketching tools:

  * Line
  * Circle
  * Arc
  * Polygon
  * Ellipse
  * Fillet
* Practiced applying accurate dimensions and constraints before creating 3D models.
* Learned how to convert a 2D sketch into a 3D model using **Extrude**.

### Design Approach

Instead of drawing the entire model at once, I learned to:

1. Analyze the shape and identify its main body (basic mass).
2. Create the primary geometry first.
3. Gradually add or remove details step by step.
4. Apply finishing features such as fillets after the main structure is complete.

### Workflow

The recommended modeling sequence is:

1. **Sketch**
2. **Extrude**
3. **Cut**
4. **Fillet**

### Skills Developed

* Analyzing complex engineering drawings before modeling.
* Choosing the correct sequence of modeling operations.
* Building models systematically without relying on direct guidance.
* Improving accuracy and efficiency in CAD modeling using Onshape.
